TCPиҝһжҺҘдёӯACK,SEQеҸҳеҢ–

д»ҘдёӢжҳҜ TCP иҝһжҺҘдёӯ ACK е’Ң SEQ дәӨдә’еҸҳеҢ–зҡ„иҜҰз»ҶзӨәдҫӢпјҲеҢ…еҗ«е®Ңж•ҙзҡ„дёүж¬ЎжҸЎжүӢгҖҒж•°жҚ®дј иҫ“е’Ңеӣӣж¬ЎжҢҘжүӢиҝҮзЁӢпјүпјҡ

рҹ“Ў иҝһжҺҘе»әз«Ӣйҳ¶ж®өпјҲдёүж¬ЎжҸЎжүӢпјү

  1. е®ўжҲ·з«Ҝ вҶ’ жңҚеҠЎеҷЁ
    SYN=1, SEQ=XпјҲйҡҸжңәеҲқе§ӢеәҸеҲ—еҸ·пјү
    е®ўжҲ·з«ҜеЈ°жҳҺиҮӘе·ұзҡ„еҲқе§ӢеәҸеҲ—еҸ· X
  2. жңҚеҠЎеҷЁ вҶ’ е®ўжҲ·з«Ҝ
    SYN=1, ACK=1, SEQ=Y, ACK=X+1
    жңҚеҠЎеҷЁзЎ®и®Өж”¶еҲ° XпјҲACK=X+1пјүе№¶еЈ°жҳҺиҮӘе·ұзҡ„еҲқе§ӢеәҸеҲ—еҸ· Y
  3. е®ўжҲ·з«Ҝ вҶ’ жңҚеҠЎеҷЁ
    ACK=1, SEQ=X+1, ACK=Y+1
    е®ўжҲ·з«ҜзЎ®и®Өж”¶еҲ° YпјҲACK=Y+1пјүпјҢеҮҶеӨҮејҖе§Ӣж•°жҚ®дј иҫ“

рҹ“Ұ ж•°жҚ®дј иҫ“йҳ¶ж®ө

sequenceDiagram

    Client->>Server: SEQ=100, Data="ABC"пјҲ3еӯ—иҠӮпјү
    Server->>Client: ACK=103пјҲSEQ=500пјҢACKеҸ·=100+3пјү
    
    Client->>Server: SEQ=103, Data="DEF"пјҲ3еӯ—иҠӮпјү
    Server->>Client: ACK=106пјҲSEQ=500пјҢACKеҸ·=103+3пјү
    
    Server->>Client: SEQ=500, Data="Hello"пјҲ5еӯ—иҠӮпјү
    Client->>Server: ACK=505пјҲSEQ=106пјҢACKеҸ·=500+5пјү

рҹ”„ дәӨдә’и§„еҲҷпјҡ

  1. жҜҸж¬ЎеҸ‘йҖҒж•°жҚ®пјҡ
    SEQ = дёҠж¬ЎSEQ + ж•°жҚ®й•ҝеәҰ
    пјҲеҰӮе®ўжҲ·з«Ҝз¬¬дёҖж¬Ў SEQ=100пјҢеҸ‘йҖҒ3еӯ—иҠӮеҗҺпјҢдёӢж¬Ў SEQ=103пјү
  2. ACKеҸ·и®Ўз®—пјҡ
    ACK = еҜ№ж–№SEQ + жҺҘж”¶ж•°жҚ®й•ҝеәҰ
    пјҲеҰӮжңҚеҠЎеҷЁж”¶еҲ° SEQ=100зҡ„3еӯ—иҠӮж•°жҚ®пјҢиҝ”еӣһ ACK=103пјү
  3. ACKеҢ…дёҚжҗәеёҰж•°жҚ®ж—¶пјҡ
    SEQ дёҚеҸҳпјҢеҸӘжҗәеёҰACKеҸ·
    пјҲзәҜACKеҢ…дёҚж¶ҲиҖ—еәҸеҲ—еҸ·пјү

рҹҡӘ иҝһжҺҘе…ій—ӯйҳ¶ж®өпјҲеӣӣж¬ЎжҢҘжүӢпјү

  1. е®ўжҲ·з«Ҝ вҶ’ жңҚеҠЎеҷЁ
    FIN=1, SEQ=110, ACK=500
    е®ўжҲ·з«ҜиҜ·жұӮе…ій—ӯпјҲFINж¶ҲиҖ—1еәҸеҲ—еҸ·пјү
  2. жңҚеҠЎеҷЁ вҶ’ е®ўжҲ·з«Ҝ
    ACK=1, SEQ=500, ACK=111
    зЎ®и®Өж”¶еҲ° FINпјҲACK=110+1пјү
  3. жңҚеҠЎеҷЁ вҶ’ е®ўжҲ·з«Ҝ
    FIN=1, ACK=1, SEQ=505, ACK=111
    жңҚеҠЎеҷЁеҸ‘еҮә FINпјҲж¶ҲиҖ—1еәҸеҲ—еҸ·пјү
  4. е®ўжҲ·з«Ҝ вҶ’ жңҚеҠЎеҷЁ
    ACK=1, SEQ=111, ACK=506
    зЎ®и®Ө FINпјҲACK=505+1пјүпјҢзӯүеҫ…2MSLеҗҺе…ій—ӯ

рҹ”Қ е…ій”®и§„еҲҷжҖ»з»“пјҡ

ж“ҚдҪң SEQ еҸҳеҢ– ACK еҸҳеҢ–
еҸ‘йҖҒSYN/FIN +1 дёҚеҸҳ
еҸ‘йҖҒж•°жҚ®пјҲNеӯ—иҠӮпјү +N дёҚеҸҳ
еҸ‘йҖҒзәҜACKеҢ… дёҚеҸҳ жӣҙж–°дёәжңҹеҫ…зҡ„дёӢдёӘSEQ
жҺҘж”¶ж•°жҚ®еҗҺACK дёҚеҸҳ =ж”¶еҲ°ж•°жҚ®жңҖеҗҺеӯ—иҠӮ+1

рҹ“Ң йҮҚиҰҒиҜҙжҳҺпјҡ
ACKеҸ·жҖ»жҳҜвҖӢвҖӢжңҹжңӣж”¶еҲ°зҡ„дёӢдёҖдёӘеӯ—иҠӮеәҸеҸ·вҖӢвҖӢпјҲеҰӮж”¶еҲ°SEQ=100зҡ„10еӯ—иҠӮж•°жҚ®пјҢиҝ”еӣһACK=110пјү
