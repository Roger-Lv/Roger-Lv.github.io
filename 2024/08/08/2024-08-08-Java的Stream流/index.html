操作符

常用中间操作符

  1. filter：用于过滤 ，接受一个返回boolean值的函数，返回一个流

  2. map：用于映射 ，映射流中的每一个元素为另一个流中的元素

  3. dinstinct: 用于去重

    教你看懂System.out::println-CSDN博客

  4. sorted: 用于排序

  5. limit: 可以将流限制为指定的元素数

常用终端操作符

  1. collect: 收集器，将流转换为其他形式

  2. forEach:遍历流

  3. Count(计数)
